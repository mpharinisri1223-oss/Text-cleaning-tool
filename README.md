# 🧹 NLP Text Cleaning Tool

## 📌 Project Overview
NLP Text Cleaning Tool is a simple Python and Streamlit project used to clean messy text step by step. The project helps users understand basic Natural Language Processing (NLP) preprocessing techniques in an interactive way.

---

## ✨ Features
- Convert text to lowercase
- Remove numbers
- Remove punctuation and special characters
- Remove extra spaces
- Step-by-step text cleaning process
- Simple and attractive dashboard
- User review section
- Sentiment analysis for reviews

---

## 🛠️ Technologies Used
- Python
- Streamlit
- NLP
- NLTK
- Pandas
- Scikit-learn
